### When to use each tool:

**Claude Code (Terminal-based):**

- Setting up new features that span multiple files
- Database migrations and schema changes
- Job queue configuration
- Initial API route structure
- "Build me a [complete feature]" tasks

**Cursor (IDE):**

- Refining UI components
- Debugging specific functions
- Adding TypeScript types
- Adjusting Tailwind styling
- Writing unit tests
- "Fix/improve this [specific code]" tasks

**v0:**

- Rapid UI component prototyping
- Learning modern React patterns
- Getting multiple design variations quickly
- Not sure how to structure a complex UI
