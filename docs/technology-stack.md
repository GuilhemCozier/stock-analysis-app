### Frontend

- **Next.js 14** (App Router) - React framework with built-in API routes
- **TypeScript** - Type safety throughout
- **Tailwind CSS** - Styling
- **shadcn/ui** - Component library
- **Server-Sent Events (SSE)** - Real-time updates from backend

### Backend

- **Next.js API Routes** - Backend endpoints
- **BullMQ** - Job queue for async processing
- **Redis** - Required for BullMQ (in-memory data store)
- **Prisma ORM** - Database interactions

### Database

- **PostgreSQL** - Main database (handles large text fields well)

### AI

- **Anthropic API** (Claude Sonnet 4.5)
- Built-in `web_search` tool (no custom MCPs needed)

### Development Tools

- **Cursor** - Main IDE for day-to-day coding
- **Claude Code** - For architectural tasks and feature scaffolding
- **v0** - UI component prototyping
- **Figma** - Design system and mockups
